Инструкция по установке

Chrome
Firefox
Edge
1

Скачайте расширение

Нажмите кнопку "Скачать для Chrome" выше и сохраните файл на ваш компьютер. После завершения загрузки распакуйте ZIP-архив в удобное место.

2

Откройте страницу расширений

Введите chrome://extensions в адресной строке браузера и нажмите Enter.

3

Включите режим разработчика

В правом верхнем углу страницы расширений переключите "Режим разработчика" в положение "Включено".

4

Загрузите распакованное расширение

Нажмите кнопку "Загрузить распакованное", выберите папку с распакованным расширением CorgPhish и нажмите "Выбрать папку".

5

Готово!

Расширение CorgPhish теперь установлено и активировано в вашем браузере Chrome. Вы можете увидеть иконку расширения в панели инструментов. Щелкните по ней, чтобы открыть настройки расширения.

1

Скачайте расширение

Нажмите кнопку "Скачать для Firefox" выше и сохраните файл на ваш компьютер. После завершения загрузки распакуйте ZIP-архив в удобное место.

2

Откройте страницу отладки

Введите about:debugging#/runtime/this-firefox в адресной строке браузера и нажмите Enter.

3

Загрузите временное дополнение

Нажмите кнопку "Загрузить временное дополнение", перейдите в папку с распакованным расширением CorgPhish, выберите файл manifest.json и нажмите "Открыть".

4

Готово!

Расширение CorgPhish теперь установлено в вашем браузере Firefox. Вы можете увидеть иконку расширения в панели инструментов. Щелкните по ней, чтобы открыть настройки расширения.

Примечание: Временные дополнения будут удалены при перезапуске Firefox. Для постоянной установки рекомендуется подождать официального релиза в Firefox Add-ons.
